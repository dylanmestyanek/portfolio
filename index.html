Macro Calculator

Description: Macro Calculator is a nutrition calculator which uses the client's information to provide a calorie, and macro nutrient goal for each day. Additionally, it divides the macro nutrients and calories by meal, or snack, in a graph format for easy consumption.

Challenges: The most challenging aspect of this project was incorporating data visualization with different forms of graphs, as well as, manipulating these graphs with the data from Redux. This was my first team-based project, which incorporated the use of AJAX and Redux together.

What I learned: This project taught me a great amount of organization and structure for larger React applications. Including the use of Redux caused the application to spread out, and required additional planning to ensure clean state management. Also, the merge conflicts that arose while working with others provided the opportunity to increase my familiarity with GitHub.

Tools used: React | Redux | Axios

Role: Front End Engineer

JavaScript Voice Timer

Description: While maintaining the standard functionality of a timer application, this vanilla JavaScript timer implements the browser's voice recognition functionality to interact with the timer.

Challenges: The largest challenge faced during this project was my first interaction with speech recognition. The majority of the time was spent dissecting the response objects from the Speech Recognition API, and discovering ways to manipulate the text into commands for the application to respond to.

What I learned: This project was one of the first JavaScript applications I worked on. The construction of this project tested my creativity and provided me with my first taste of the JavaScript foundation, such as: DOM manipulation and Array/Object methods. The experience of building this app taught me patience, persistence, and planning.

Tools used: HTML | CSS | JavaSript

Role: Front End Engineer

Spotifier Song Suggester

Description: Spotifier was a group project built to mimic the functionality of Spotify. Upon creating an account, the user is provided the option to look up songs from a library, save those songs to their user-specific playlist, and view 10 similar tracks related to that song.

Challenges: Being my first group-oriented front-end project, it took some patience to comprehend "efficient routing". There was a lot of trial and error when it came to building out the application components, and how to implement a convenient user flow throughout the app.

What I learned: This project taught me effective team communication and collaboration, as well as, allowed me to incorporate all of my current front-end knowledge into one project. I gained an appreciation for React, and hands-on experience with making API calls. Working on this project unveiled the unlimited possibilities of front-end creativity.

Tools used: React | Redux | Axios | Sass | Formik | Yup

Role: Front End Engineer

Weight Lifting Journal API

Description: This Weight Lifting Journal project is an API built that allows a front-end application to connect, and retrieve the user stored in the database and display their workout data. This API provides endpoints for all CRUD functionality the application may need to add information about their workouts and exercises.

Challenges: This was my first back-end group project, built with one other developer. The largest barrier we faced was planning an efficient database schema, and envisioning how our endpoints would be built through out the project. We faced a handful of challenges when troubleshooting errors, and had to patiently test each endpoint.

What I learned: Building a fully functioning API gave me a solid understanding of how the front and back-end could connect together. It allowed me to pinpoint prior front-end errors I would receive, and helped me see the importance of informative, precise documentation for each endpoint. Additionally, I gained great appreciation for clear and concise API error handling.

Tools used: Node | Express | Knex | PostgreSQL | JsonWebToken

Role: Back End Engineer
